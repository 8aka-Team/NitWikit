## Windows环境

### 1、下载二进制文件

前往 PMMP 专门打包 PHP 的Github仓库下载对应系统的 PHP 文件：[Github Releases](https://github.com/pmmp/PHP-Binaries/releases)

### 2、启动下载的二进制文件

在 `bin` 文件夹中，双击 `vc_redist.x64.exe` ，接受条款和条件，然后安装。这将安装 PHP 二进制文件所需的 Microsoft Visual C++ Redistributable
