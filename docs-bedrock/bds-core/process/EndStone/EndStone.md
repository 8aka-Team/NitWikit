# EndStone

EndStone 提供类似 Bukkit 插件开发者的友好 API，简化了低层系统 API 和挂钩的处理。可以使用 C++ 或 Python 编写插件。

EndStone的很多插件可以在 **[minebbs](https://www.minebbs.com/resources/categories/bdserver.38/)** 和 **[PyPI](https://pypi.org/search/?q=endstone)** 上查找了。
