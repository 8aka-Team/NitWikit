# 资源包

:::tip

这一部分的教程不是教你如何**制作**资源包,而是教你如何**托管/压缩/混淆**资源包!!

:::

## 注意

JAVA 版的资源包分发与基岩版不同，它是云分发的，提供给客户端的应该是一个**直链**，而不是资源包本身

不管你用什么方式，都必须是直链才可正常使用！

**什么是直链？**

> 直接从服务器下载数据

不需要登陆账号，不需要点击链接打开网页才下载
