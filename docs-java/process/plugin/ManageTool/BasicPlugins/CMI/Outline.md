## 前置

CMI 从 9.x 版本开始需要 CMILib。你可以从 [这里](https://www.spigotmc.org/resources/cmilib.87610/) 下载。

## 下载

在 [SpigotMC](https://www.spigotmc.org/resources/3742/) 帖内自行寻找。

## 安装

将 `CMILib1.5.0.8.jar` 和 `CMI-9.7.4.6.jar` 放入 `\plugins` 文件夹下后重启服务器即可。
