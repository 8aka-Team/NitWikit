可以在语言文件中使用 PlaceholderAPI 变量来使CMI发送信息的时候显示一些东西。

只有以 `%cmi_` 开头的变量会被 PlaceHolderAPI 识别到,  但是在大多数地方只需 CMI 本体即可使用。

执行 `/cmi placeholders` 命令可以查看目前所有的静态变量。
