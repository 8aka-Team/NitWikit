## 插件不能做的事

例如（无客户端 mod 的情况）：

- 插件不能做自定义新的按键(原版仅能监听 Shift,F,Q 共四种组合)
- 获取玩家视角（第一人称，第二人称，第三人称）
- 不靠资源包或 mod 什么的不能修改计分板的背景
- 不大可能检测和拦截灵魂出窍

总之，一些纯客户端的行为，插件是不能控制的。
